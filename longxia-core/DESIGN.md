## 布局
- 最大内容宽度: 1200px
- 页面左右内边距: 24px (移动端) / 48px (桌面端)
- 区块间距: 96px (桌面) / 64px (移动端)
- 圆角: 20px (大卡片) / 12px (按钮) / 999px (全圆角)
- 响应式断点: 768px (平板) / 1024px (桌面)

## 动效规范
- 过渡曲线: cubic-bezier(0.32, 0.72, 0, 1) (苹果弹簧动效)
- 过渡时长: 300ms (常规) / 700ms (入场动效)
- 特殊动效:
  - 背景渐变缓慢流动效果 (20s 循环)
  - 卡片悬停上浮 + 光晕效果
  - 滚动时元素淡入上移
  - 按钮悬停时的微缩放 + 光晕
  - 玻璃态元素的微光扫过效果
